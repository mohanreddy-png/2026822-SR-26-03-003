# ICLMDL-SED
## Integration of Corpus Linguistics and Multi-modal Deep Learning for Enhanced Semantic-Driven Emotion Detection on Textual Data

This repository contains the official implementation of the proposed **ICLMDL-SED** framework for semantic-driven emotion detection from textual data using corpus linguistics and deep learning techniques.

The proposed framework integrates:

- Text preprocessing techniques
- Word2Vec embedding
- Temporal Convolutional Network (TCN)
- Bidirectional Long Short-Term Memory (BiLSTM)
- Self-Attention Mechanism (SAM)

to accurately classify emotions from textual datasets.

# Abstract

Emotion detection from textual data plays a crucial role in natural language processing, sentiment analysis, human–computer interaction, and intelligent systems. This work proposes an Integrated Corpus Linguistics and Multi-modal Deep Learning-based Semantic-Driven Emotion Detection (ICLMDL-SED) framework for accurate emotion classification from textual inputs. The proposed method incorporates preprocessing techniques such as tokenization, stop-word removal, stemming, and POS tagging to enhance contextual representation. Word2Vec embeddings are utilized for semantic feature extraction, while a hybrid TCN-BiLSTM-SAM architecture performs effective emotion classification. Experimental evaluation on benchmark emotion datasets demonstrates superior performance compared to existing approaches.
